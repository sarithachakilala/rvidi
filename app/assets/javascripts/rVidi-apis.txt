1. USER  : 
Register, Login and Creating an User to rvidi, via rvidi app or facebook or twitter, updating user information.
Required fields: first_name, last_name, email, username, password, password_confirmation, terms_conditions

2. AUTHENTICATION : 
Adding facebook user details and twitter details
Required fields: ouath_token_secret, oauth_expires_at, uid, user_id


3. SHOW :
Creating, Updating & editing show
Required fields: title, description, display_preferences, contributor_preferences, display_preferences_password, contributor_preferences_password, duration, cameo_duration


4. CAMEO : 
Creating, Updating, editing, deleting cameo and uploading to kaltura.
Required fields: director_id, user_id, status, name, thumbnail_url, download_url



5. COMMENT:
Creating comments to show
Required fields: body,show_id,user_id,user_name

6. NOTIFICATION: 
Creating when a cameo is added, friend request, invitations to contribute, follower contributions
Required fields: from_id, to_email, to_id


7. INVITE FRIEND : 
Creating invitations for a show via email, updating

8. FRIEND MAPPING: 
Creating, Updating when friend mapping when accepted, rejected.
